# 📷 Blue--App 📷

## 📱 Sobre o Projeto

O **Blue--App** é um aplicativo mobile desenvolvido em **React Native** com **Expo**, criado para realizar a leitura de QR Codes utilizando a câmera do dispositivo.

O aplicativo possui uma interface simples, moderna e intuitiva, permitindo que o usuário escaneie códigos QR rapidamente diretamente pelo celular.

---

## 🚀 Tecnologias Utilizadas

* React Native
* Expo
* JavaScript
* Expo Camera / QR Scanner
* StyleSheet (CSS-in-JS)

---

## 🎨 Funcionalidades

* 📷 Leitura de QR Code em tempo real
* 📱 Interface mobile responsiva
* 🎯 Scanner centralizado na tela
* ⚡ Abertura rápida da câmera
* 🌙 Interface escura moderna
* 🔄 Navegação entre telas

---

## 📸 Preview do Aplicativo

O aplicativo apresenta:

* Tela inicial simples
* Área para iniciar o scanner
* Scanner de QR Code utilizando a câmera do celular
* Interface intuitiva para facilitar a leitura

## ⚙️ Como Executar o Projeto

### 1️⃣ Clone o repositório

```bash
git clone <URL_DO_REPOSITORIO>
```

### 2️⃣ Acesse a pasta do projeto

```bash
cd Blue--App
```

### 3️⃣ Instale as dependências

```bash
npm install
```

### 4️⃣ Execute o projeto

```bash
npx expo start
```

---

## 📲 Executando no Celular

1. Instale o aplicativo **Expo Go** no celular.
2. Execute o comando:

```bash
npx expo start
```

3. Escaneie o QR Code exibido no terminal.
